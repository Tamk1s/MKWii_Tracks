1. What is it?

A common.szs item pack, with Microsoft Hover! (1995)/Windows 95-themed items.
It is a part of Hover Pack (http://wiki.tockdom.com/wiki/Hover_Pack),
a distribution of battle tracks from Microsoft Hover.

This itempack changes the items as such:

==================

2. Installation

In order to install and use the itempack, just place the SZS archive files into the "My Stuff"
folder of your favorite distribution and run it through Riivolution.==================
